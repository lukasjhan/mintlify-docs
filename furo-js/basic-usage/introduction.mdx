## Furo JS

Furo JS는 웹 프레임워크 어디서든 Furo의 기능을 사용할 수 있는 통합 JavaScript SDK입니다.<br/>
React를 비롯하여 Vue, Svelte, Plain HTML 등 어디서든 Furo의 기능을 간편하게 통합하고 사용하실 수 있습니다.

Furo.js의 API를 사용하여 추가적인 구현없이도 내 웹사이트에 로그인 UI를 표시하고, 유저 정보를 조회하는 등의 작업을 쉽게 할 수 있습니다.
