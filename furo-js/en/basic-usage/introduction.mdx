## Furo JS

Furo JS is an integrated JavaScript SDK that enables the use of Furo's features in any web framework.<br/>
You can easily integrate and use Furo's capabilities with various technologies including React, Vue, Svelte, and Plain HTML.

Using the Furo.js API, you can effortlessly display a login UI on your website and perform tasks like fetching user information without the need for additional implementations.
