---
title: 'User Authenticate'
api: 'POST /sessions/code/authenticate'
description: 'Authenticate a user with a code'
---

### Header

<ParamField header="origin" type="string" required>
  The origin of the request. This is the URL of the page that the user is
</ParamField>

### Body

<ParamField body="code" type="string" required>
  The code that login request returns.
</ParamField>

### Response

<ResponseField name="access_token" type="string">
  JWT access token.
</ResponseField>

<ResponseField name="refresh_token" type="string">
  JWT refresh token.
</ResponseField>
