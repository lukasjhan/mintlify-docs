---
title: 'OTP Auth'
api: 'POST /magic_links/authenticate'
description: 'Start magic link login flow'
---

### Body

<ParamField body="requestId" type="string" required>
  Request ID of the magic link request.
</ParamField>

<ParamField body="code" type="string" required>
  OTP code sent to the user.
</ParamField>

## Response

<Warning>
  In response, property access_token and refresh_token is deprecated. Use
  **code** property instead.
</Warning>
